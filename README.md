# BookBot

Bot Telegram sinh ebook (EPUB 3) bằng tiếng Anh, đạt chuẩn Apple Books.

## Deploy trên Railway

1. Fork repo, connect Railway.
2. Thêm biến môi trường `TELEGRAM_TOKEN`, `OPENAI_API_KEY`.
3. Bấm **Deploy**.

Trong Telegram:
```
/newbook How to use GoLogin
```
Bot trả file .epub ~1‑2 phút.
